# Contact Book (Python)

## Description
A simple Python Contact Book program that lets users store and manage contacts using a dictionary.

## Features
- Add contact
- View contacts
- Search contact
- Delete contact
- Edit contact
- Exit program

## How It Works
The program uses a menu-driven system where the user selects options to manage contacts.

## Technologies Used
- Python
- Dictionary
- Functions
- Loops
- Exception Handling
